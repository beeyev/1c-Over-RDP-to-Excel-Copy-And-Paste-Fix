# Решение проблемы при копировании из 1С (RDP, Терминал) в локальный Excel.

Проблема возникает начиная с Windows 10 билд 1607  и Windows Server 2016 билд 1607 .
Затрагивает только копирование непосредственно из 1С, запущенной удаленно, в Excel, запущенный локально. Неправильно копируются формулы и числа, сдвигаются ячейки.

Для Windows Server 2016 заменяем файл rdpclip.exe на версию из Windows Server 2012 R2.
Для клиентов под Windows 10 меняем файлы mstsc.exe и mstscax.dll на версии из Windows 10 билд 1511.

Готовые файлы для [сервера](server) и [клиента](client).
